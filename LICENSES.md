# Licensing Notes

This monorepo currently contains multiple first-party components with package-specific
license files and metadata.

Current workspace licensing:

- `hyperpipe-core`: Apache-2.0
- `hyperpipe-core-host`: Apache-2.0
- `hyperpipe-bridge`: Apache-2.0
- `hyperpipe-tui`: Apache-2.0
- `hyperpipe-gateway`: Apache-2.0
- `hyperpipe-desktop`: MIT

The monorepo intentionally contains a mixed licensing model today. Refer to the
package-level metadata and license files that ship with each workspace instead of
assuming a single monorepo-wide license.
